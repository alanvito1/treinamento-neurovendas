# Caminho do Fechamento - Treinamento de Elite

Este é o repositório do curso **Caminho do Fechamento**, um treinamento focado em transformar vendedores iniciantes de consórcio em **NeuroClosers de Alta Performance**.

O projeto foi construído como uma aplicação web estática híbrida, combinando conceitos de Neurovendas, PNL e técnicas agressivas de fechamento.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estrutura das páginas e players de vídeo/PDF.
- **Vanilla CSS**: Design System premium com efeito Glassmorphism, Dark Mode e cores focadas em conversão (Âmbar/Dourado).
- **JavaScript**: Lógica para controle de progresso (em expansão).
- **Vercel**: Hospedagem e deploy contínuo.

## 📁 Estrutura do Projeto

O projeto segue uma organização moderna e limpa:

- `index.html`: Hub central do aluno.
- `introducao.html`: Boas-vindas e introdução ao curso.
- `modulo1.html` a `modulo5.html`: Páginas de conteúdo de cada módulo.
- `roleplay.html`: Laboratório prático com perfis de clientes para treinamento.
- `conclusao.html`: Resumo final e plano de ação.
- `style.css`: Arquivo central de estilos.
- `app.js`: Arquivo central de scripts.
- `caminho_do_fechamento/`: Pasta contendo os materiais pesados (PDFs, PPTs e Vídeos) organizados por módulo.

## 💻 Como Executar Localmente

1. Clone o repositório (ou baixe os arquivos).
2. Abra o arquivo `index.html` diretamente em qualquer navegador moderno.
3. Não há necessidade de servidor backend (Node.js, etc.) para visualização básica.

## 🌐 Deploy

O projeto está configurado para deploy na Vercel.
