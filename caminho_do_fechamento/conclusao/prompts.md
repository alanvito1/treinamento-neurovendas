# Prompts: Considerações Finais e Resumo

Cole estes prompts no chat do NotebookLM para gerar o conteúdo de encerramento do curso.

## 1. Prompt para Texto de Estudo (PDF)
```text
Com base nas fontes e em tudo o que estruturamos, elabore um texto de "Considerações Finais e Resumo" para o curso "Caminho do Fechamento".
O objetivo é fazer uma recapitulação emocionante e motivadora de todo o aprendizado (desde a mente do NeuroCloser até o fechamento).
Gere também um pequeno "Plano de Ação para os primeiros 30 dias" do vendedor iniciante.
O tom deve ser de "mestre para aluno", incentivando a execução imediata.
```

## 2. Prompt para Roteiro de Slides
```text
Crie uma estrutura de slides para a aula de encerramento.
Foque em um checklist do que o aluno aprendeu e os próximos passos para ele faturar alto com consórcio.
```

## 3. Prompt para Vídeo de Despedida/Resumo
```text
Crie um roteiro de vídeo de encerramento (2 a 3 minutos) onde o instrutor parabeniza o aluno por concluir o curso, resume o poder do método e dá a última palavra de incentivo para ele ir para o campo e fechar vendas.
```
