# Prompts: Introdução do Curso

Cole estes prompts no chat do NotebookLM para gerar o conteúdo introdutório.

## 1. Prompt para Texto de Boas-Vindas e Visão Geral
```text
Com base nas fontes, elabore um texto de introdução para o curso "Caminho do Fechamento".
O foco deve ser acolher o vendedor iniciante de consórcio, explicando a transformação (de inseguro a NeuroCloser de Alta Performance).
Apresente a visão geral do curso e os 5 módulos que serão estudados.
O tom deve ser motivador, profissional e focado em resultados.
```

## 2. Prompt para Slides da Aula Inaugural
```text
Crie uma estrutura de tópicos para slides de uma aula inaugural do curso "Caminho do Fechamento".
Aborde a Visão Geral do Curso e a Transformação prometida.
Deixe o conteúdo visual e direto ao ponto.
```

## 3. Prompt para Roteiro de Vídeo Resumo
```text
Crie um roteiro de vídeo curto (máximo 2 minutos) para o instrutor dar as boas-vindas aos alunos e explicar rapidamente o que eles vão aprender no curso.
```
