# Prompts: Módulo 3 - Sondagem Estratégica

Cole estes prompts no chat do NotebookLM para gerar o conteúdo do Módulo 3.

## 1. Prompt para Texto de Estudo (PDF)
```text
Com base nas fontes, elabore o conteúdo textual completo para o "Módulo 3: Sondagem Estratégica e Escuta Ativa".
O foco é fazer o cliente falar e descobrir sua real necessidade.
Aborde os seguintes tópicos em profundidade:
- Escuta Ativa (regra 80/20).
- O Método DTC (Destino, Tempo e Condições).
- Diferença entre Dor e Desejo na venda de consórcio.
Traga exemplos de perguntas de sondagem que revelam as objeções ocultas.
```

## 2. Prompt para Roteiro de Slides
```text
Com base no conteúdo do Módulo 3, crie uma estrutura de slides.
Destaque o Método DTC e a matriz de Dor vs. Desejo de forma visual.
```

## 3. Prompt para Vídeo Resumo
```text
Crie um roteiro de vídeo de resumo (3 minutos) para o Módulo 3, explicando como fazer uma sondagem que desarma o cliente e prepara para o fechamento.
```
