# Prompts: Módulo 5 - Fechamento e Fidelização

Cole estes prompts no chat do NotebookLM para gerar o conteúdo do Módulo 5.

## 1. Prompt para Texto de Estudo (PDF)
```text
Com base nas fontes, elabore o conteúdo textual completo para o "Módulo 5: O Fechamento Sem Pressão e a Fidelização Eterna".
O foco é assumir o controle, fechar a venda e gerar indicações.
Aborde os seguintes tópicos em profundidade:
- Fechamento Direto (dar opções de fechamento).
- Perdendo o medo de pedir o RG/CPF.
- Fluxo de Cadência para recuperar clientes sumidos (vácuo).
- Como pedir indicações no ato da assinatura.
- Mapa de recompra e retenção do cliente.
Traga os scripts de fechamento e de pedido de indicações.
```

## 2. Prompt para Roteiro de Slides
```text
Com base no conteúdo do Módulo 5, crie uma estrutura de slides.
Destaque os scripts de fechamento e o fluxo para recuperar clientes que sumiram.
```

## 3. Prompt para Vídeo Resumo
```text
Crie um roteiro de vídeo de resumo (3 minutos) para o Módulo 5, focando na atitude do fechamento e no poder de gerar indicações imediatas.
```
