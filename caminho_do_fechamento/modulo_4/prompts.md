# Prompts: Módulo 4 - Apresentação de Valor e MFA

Cole estes prompts no chat do NotebookLM para gerar o conteúdo do Módulo 4.

## 1. Prompt para Texto de Estudo (PDF)
```text
Com base nas fontes, elabore o conteúdo textual completo para o "Módulo 4: Apresentação de Valor e Método MFA".
O foco é apresentar o projeto matando objeções e o leilão de preços.
Aborde os seguintes tópicos em profundidade:
- O fim das propostas em PDF pelo WhatsApp.
- Focando no Resultado (vender a solução, não a carta).
- Aplicação prática do Método MFA (Método de Fechamento Antecipado).
- Criação de Urgência e Escassez Reais.
Traga exemplos de diálogos usando o MFA.
```

## 2. Prompt para Roteiro de Slides
```text
Com base no conteúdo do Módulo 4, crie uma estrutura de slides.
Destaque o passo a passo do Método MFA e como apresentar valor antes do preço.
```

## 3. Prompt para Vídeo Resumo
```text
Crie um roteiro de vídeo de resumo (3 minutos) para o Módulo 4, focando na técnica do Fechamento Antecipado e em como não perder a venda por preço.
```
