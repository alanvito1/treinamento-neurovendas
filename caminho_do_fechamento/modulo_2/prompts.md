# Prompts: Módulo 2 - A Máquina de Atração

Cole estes prompts no chat do NotebookLM para gerar o conteúdo do Módulo 2.

## 1. Prompt para Texto de Estudo (PDF)
```text
Com base nas fontes, elabore o conteúdo textual completo para o "Módulo 2: A Máquina de Atração".
O foco é ensinar o aluno a prospectar ativamente e prender a atenção.
Aborde os seguintes tópicos em profundidade:
- O Hábito de Prospectar (20 a 25 contatos diários, foco em ligação).
- Abordagem sem falar a palavra "Consórcio" no início.
- Os Gatilhos Iniciais (reciprocidade e prova social).
Traga scripts práticos de abordagem baseados no conteúdo das mentoras.
```

## 2. Prompt para Roteiro de Slides
```text
Com base no conteúdo do Módulo 2 (Prospecção), crie uma estrutura de slides.
Destaque os scripts de abordagem e as regras de ouro para prospectar em qualquer lugar.
```

## 3. Prompt para Vídeo Resumo
```text
Crie um roteiro de vídeo de resumo (3 minutos) para o Módulo 2, focando na importância da prospecção ativa e no exemplo de um script de abordagem que funciona.
```
